Welcome to RPO UX

Each of these applications cover a different aspect of the overall RPO workflow.


Oracle is a real-time intel database and CONOPS analysis tool to ensure Space Ops missions and intel come with the most up to date decisions.


Phalanx is for Space Policy makers to simulate "what-if" scenarios before committing to a policy change.


Apex is an RPO trajectory viewer and mission planner, complete with playback tools and ODCIL.


Warden is an SDA awareness, anomaly detection, and threat response tool with rich trajectory visualization.
